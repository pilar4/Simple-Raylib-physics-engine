# PhysicsEngine
**Simple single-threaded 2D physics engine, developed from scratch using RayLib and C++**  
**This project is designed to help learn and explore the fundamentals of simulating physics.**
### Planned:
- [x] Bouncing balls
- [ ] Rigid bodies
- [ ] Sand
- [ ] Water / other liquids
- [x] Gravity manipulation
- [x] Time slow / stop
- [ ] Toolbar
- [ ] Some basic settings
- [ ] Explosions
